# La Candona Website

Este es el repositorio oficial para la página web de La Candona, una agencia de desarrollo web profesional en Guatemala.

## Tecnologías

- React
- Vite
- Tailwind CSS
- Lucide React (Iconos)

## Instalación

1.  Clonar el repositorio
2.  Instalar dependencias: `npm install`
3.  Correr servidor de desarrollo: `npm run dev`
